## General info
This project is a simple example of the REST API for a user-role relationship.

## Technologies
* Spring Boot (2.2.4 released)
* MySQL

## Requirements
* JDK 8
* Maven 3

## Setup
1) Build Spring Boot Project with Maven

    `mvn install`
2) Run Spring Boot App with Maven

    `mvn spring-boot:run`
